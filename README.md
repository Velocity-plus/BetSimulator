# BetSimulator
A betting simulator, if you doing automatic betting Martin Gale method. 

# How-To-Use:
Open the .py file and in the beginning of the file, you will see:
""" Settings """
Bankroll     = "0.00033300$"
Starting_Bet = "0.00000002$"

Bets = "5184000"
Win_Chance = "49.5%"

Multiply_On_Lose = 2
Multiply_On_Win = 1

Change them to your needs and run the program.
